# SkyPay Platform Documentation

## Architecture Overview

SkyPay is a hybrid financial platform that bridges traditional finance (TradFi) with decentralized finance (DeFi) through a regulated, institution-grade infrastructure.

### Core Components

1. **Backend API** - Node.js/TypeScript REST API with Express
2. **Smart Contracts** - Solidity contracts on Polygon
3. **Web SDK** - TypeScript SDK for frontend integration
4. **Web Application** - Next.js React application
5. **Mobile Application** - React Native cross-platform app
6. **Infrastructure** - AWS EKS, RDS, Redis, Terraform
